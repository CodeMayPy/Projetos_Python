## 🐍 Projetos_Python

Este repositório contém um conjunto de mão na massa de projetos pessoais meus, onde prático conceitos de lógica de programação, manipulação de strings e automação.

---
# 1- Verificador de senha: 🛠️ Em desenvolvimento
<br>

Um script que analisa a força de senhas baseada em critérios de segurança corporativa (comprimento, caracteres especiais e previsibilidade).
* **O que faz:** Solicita uma senha e realiza uma análise em tempo real para verificar se ela atende aos requisitos mínimos de segurança.
* **Conceitos aplicados:** any() & len(): utilizado para criar filtros inteligentes que verificam a existência dos caracteres desejados.
**Foco:** Prevenção de ataques de Força Bruta (Brute Force) e aumento da entropia das credenciais.

# 2- Bot de análise de preços: 
<br> 

**(Em Desenvolvimento)**
Automação focada em Web Scraping e análise de dados para tomada de decisão financeira.
**O que faz:** Pega o dado que você deseja acompanhar, pergunta se quer receber um alerta e caso queira ele o envia por email.
* **Conceitos aplicados:** Web Scraping com Selenium: Manipulação de WebDrivers para automação de browser e coleta de dados dinâmicos.
**Foco:** Automação de processos web e extração de dados (Data Crawling).

# 3- Gerenciador de Senhas:
<br>

**(Em Desenvolvimento)**
Um sistema de persistência de credenciais que foca no "Zero Trust". 
* **O que faz:** Solicita uma senha, gera um hash criptográfico seguro (SHA-256) e armazena em um banco de dados local.
* **Conceitos aplicados:** Manipulação de arquivos (I/O), tratamento de bytes, criptografia de sentido único (Hashing) e segurança de dados.
* **Tech Stack:** Python, `hashlib`.
